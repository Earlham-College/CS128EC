To create an app.

1) Find a .icns logo. E. g. https://www.iconfinder.com/search/?q=science&price=free
2) You should create the code, such as testApp.py
3) Go to setup.py python code and change everything you need to update, such as Python code name, icon name and path, etc.
    - To install py2app in mac
        - On MAC or Linux: pip3 install py2app
        - On Windows with Anaconda: conda install
4) The executable file will be on dist directory.
